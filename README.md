# MuscleMate

_**MuscleMate** é um diário de treinos que permite registrar treinos, exercícios e sets de forma fácil e organizada._

## Funcionalidades Principais

### Views
- **HomeView**: Lista de treinos filtrada por data.
- **AddWorkoutView**: Modal para criar um novo treino.
- **AddExerciseView**: Modal para adicionar um exercício.
- **AddSetView**: Modal para adicionar sets.
- **WorkoutDetailView**: Lista detalhada de exercícios e sets do treino.

### App Intents
- **WorkoutIntentView**: Cria treinos em uma data específica usando App Intents.

## Views
### HomeView
**Funcionalidades**
- Calendário semanal para seleção de treinos.
- Lista de treinos com nome e quantidade de exercícios.
- Botão "+" para criar novos treinos.
